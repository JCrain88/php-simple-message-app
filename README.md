# Simple Messaging Application (HTML/CSS/JS)

A client-side messaging application built with vanilla HTML, CSS, JavaScript, and Bootstrap 5. All data is stored in browser's localStorage.

## Features
- User registration and login system
- Send direct messages to other users
- Create and join group chats
- View message history
- Persistent data using localStorage

## Requirements
- Modern web browser with localStorage support
- No server or backend required!

## Installation Steps

1. **Download the files**:
    - `index.html`
    - `app.js`

2. **Open in browser**:
    - Simply double-click `index.html` or open it in your browser
    - Or serve it with a local server: `python -m http.server 8000`
